# ニシノ設備 見積作成システム

Excel VBAで作成された見積作成ソフトをWebアプリへ移行するためのNext.js実装です。

## 主な機能

- 見積作成、編集、閲覧
- 給水、排水、給湯、衛生、空調、換気、消火、ガス、その他の工種別明細
- 顧客ごとの上乗せ係数
- 材料費 + 労務費 + 経費を基準にした単価計算
- 現行通りの諸経費計算: 直接工事費 × 15%、百円単位切り捨て
- 消費税は現行通り「別途申し受けます」表記
- 鏡と各種内訳明細を1つの印刷/PDF画面に出力
- 各種マスタ管理
- PDFから単価マスタを解析し、差分確認後に反映
- 管理者/一般ユーザーの権限制御

## 単価PDFの想定形式

東京都の機械設備工事積算実務マニュアルのような、以下の並びを想定しています。

```text
頁 名称 摘要 備考 施工 単位 材料単価 材料費 労務費 経費 複合単価
```

日本語PDFを読み取るため、PDF.js用のCMapを `public/cmaps/` に配置しています。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## Supabase

`supabase/schema.sql` をSupabase SQL Editorで実行してください。

初期版はlocalStorageで動作します。Supabase本接続は `.env.local` を設定し、`lib/store.ts` をDB読み書きへ置き換えて進めます。
